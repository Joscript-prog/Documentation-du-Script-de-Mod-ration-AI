Documentation du Script de Modération AI

Ce script permet une modération avancée des commentaires en utilisant plusieurs modèles d'intelligence artificielle combinés avec des listes de mots-clés statiques et dynamiques.

Version Actuelle

v7.6.3 - Dernière mise à jour: 27/05/2024

Installation

Pour installer le script, suivez ces étapes :

  1. Installez l'extension Tampermonkey sur votre navigateur
  2. Cliquez sur le lien d'installation du script
  3. Confirmez l'installation dans Tampermonkey
  4. Ouvrez la page de modération pour voir l'interface s'afficher
// Exemple de configuration du script // ==UserScript== // @name Modération AI // @namespace http://tampermonkey.net/ // @version 7.6.3 // @description Modère les commentaires avec IA // @author VotreNom // @match https://moderatus.netino.com/* // @grant GM.xmlHttpRequest // @grant GM.addStyle // @require https://cdn.jsdelivr.net/npm/@tensorflow/tfjs@4.15.0/dist/tf.min.js // ==/UserScript==

Fonctionnalités Principales

Le script offre plusieurs fonctionnalités avancées :

  • Analyse des commentaires avec plusieurs modèles IA
  • Détection de toxicité en temps réel
  • Système de picking pour exporter les commentaires
  • Gestion des mots-clés dynamiques
  • Entraînement personnalisé du modèle
  • Export des données en Excel

Universal Sentence Encoder

Système de Picking

Le système de picking permet de sélectionner des commentaires pour les exporter :

Fonctionnalités du Picking

  • Sélection multiple de commentaires
  • Stockage local des commentaires pickés
  • Export en Excel avec mise en forme
  • Interface utilisateur intuitive

Comment utiliser le Picking

  1. Cliquez sur le bouton "PICK" à côté d'un commentaire
  2. Accédez à la sélection via le bouton "Process PICK"
  3. Exportez les commentaires sélectionnés en Excel

Gestion des Mots-clés

Le système de mots-clés permet d'ajouter des règles de modération personnalisées :

Types de Mots-clés

  • Mots simples : Détection de mots individuels
  • Ensembles : Détection de combinaisons de mots
  • Statiques : Liste prédéfinie de mots sensibles
  • Dynamiques : Mots ajoutés par l'utilisateur

Interface de Gestion

L'interface permet d'ajouter, modifier et supprimer des mots-clés :

  • Ajout par lot avec séparation par virgules
  • Import/Export des listes au format JSON
  • Visualisation des mots-clés actifs

Données d'Entraînement

Le système collecte des données pour améliorer le modèle personnalisé :

Comment marquer un commentaire

  1. Cliquez sur "☠ Toxique" pour marquer un commentaire comme toxique
  2. Cliquez sur "✓ OK" pour marquer un commentaire comme sain
  3. Accédez aux données via "Données d'entraînement"

Export des Données

Plusieurs options d'export sont disponibles :

Formats d'Export

  • Excel : Format tableur avec colonnes
  • CSV : Format texte compatible
  • JSON : Pour les mots-clés et données

Export Excel

L'export Excel inclut toutes les métadonnées des commentaires :

  • ID, Origine, Statut
  • Opérateur, Dates
  • Pseudo, Commentaire

Performances

Le script est optimisé pour des performances optimales :

Optimisations

  • Chargement asynchrone des modèles
  • Analyse par lots pour réduire la charge
  • Stockage local pour éviter les requêtes répétées
  • Interface non bloquante
